📚 ICP Library Management System
Official Desktop Solution for Imperial College of Pharmacy

📝 Project Overview
This is a professional desktop application designed to digitize and simplify the management of college library operations. It replaces manual paperwork with a fast, reliable, and user-friendly digital interface.

What it does:

Inventory Tracking: Manage a complete digital database of all books in the library.

Student Management: Track student library records and borrowing history.

Transaction System: Quickly issue and return books with automated status updates.

Professional Reporting: Generate and export library data directly to Excel and PDF for college records.

How it works (Technology Stack):

Frontend: Built with Flutter (Dart) for a modern, responsive user interface.

Database: Uses SQLite for high-speed, local data storage (no internet required).

File Services: Uses specialized services to handle automated Excel/PDF generation.

Distribution: Packaged into a Windows Executable using Inno Setup.
