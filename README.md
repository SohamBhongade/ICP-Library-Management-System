# 📚 ICP Library Management System
> **Official Desktop Solution for Library Management**

---

## 📝 Project Overview
This application is a professional desktop solution designed to digitize and simplify the management of library operations. It replaces manual paperwork with a fast, reliable, and user-friendly digital interface.

### ✨ What it does
* **Inventory Tracking:** Manage a complete digital database of all books in the library.
* **Student Management:** Track student library records and borrowing history.
* **Transaction System:** Quickly issue and return books with automated status updates.
* **Professional Reporting:** Generate and export library data directly to **Excel** and **PDF** for official college records.

### 🛠️ Technology Stack
* **Frontend:** Built with **Flutter (Dart)** for a modern UI.
* **Database:** Uses **SQLite** for high-speed, local data storage.
* **File Services:** Specialized services for automated **Excel/PDF** generation.
* **Distribution:** Packaged into a Windows Executable using **Inno Setup**.

---

## 🚀 How to Install
You don't need any coding knowledge to use this app. Follow these 3 simple steps:

1.  **Download:** Click the link below to get the setup file:
    👉 **[Download ICP_Library_Setup.exe](https://github.com/SohamBhongade/ICP-Library-Management-System/raw/refs/heads/main/ICP_Library_Setup.exe)**
2.  **Run:** Open the downloaded file and follow the on-screen instructions.
3.  **Launch:** Look for the **ICP Library System** icon on your Desktop and double-click to start!
